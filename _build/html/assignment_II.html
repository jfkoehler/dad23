Functions#

In this section, every question gives example(s). You can (but don‚Äôt need to) name your function accordingly. You should test out the example to make sure your function works properly. If the example doesn‚Äôt work, your answer is likely incorrect.

  1. Write a function to compute the area of a triangle. You only need the base and height of a triangle to compute its area.

  • Example: area_triangle(5, 4) == 10.0

  1. Write a function that takes in a number max_num and returns the sum of the numbers 1 through max_num (inclusive).

  • Example: sum_n(5) == 1 + 2 + 3 + 4 + 5 == 15

  1. Write a function that takes in a number max_num and returns the sum of the square of the numbers 1 through max_num (inclusive).

  • sum_squares(5) == 1 + 4 + 9 + 16 + 25 == 55

  1. Write a function that takes a number (an amount in euros) and convert it to U.S. dollars. There are 1.1 U.S. dollars per euro.

  • to_dollar(10) == 11.0

  1. Write a function that takes a list of numbers (amounts in euros), and returns them converted to dollars. Use your answer to the previous question in your answer to this question.

  • Example: to_dollar_list([1, 10, 100]) == [1.1, 11.0, 110.0]

  • Note: You may get some decimal weirdness here.

  1. Write a function that takes a number (a person‚Äôs height) and returns ‚ÄúTall enough‚Äù if they are at least 5 feet (60 inches) or ‚ÄúToo short‚Äù if they are below 5 feet.

  • Example: is_tall(72) == 'Tall enough'

  • Example: is_tall(52) == 'Too short'

  1. Write a function that takes a list of numbers (peoples‚Äô heights) and returns a list of strings ‚ÄúTall enough‚Äù or ‚ÄúToo short‚Äù as in the previous problem. Use your answer to the previous problem to help answer this problem.

  • Example: is_tall_list([58, 70, 60]) == ['Too short', 'Tall enough', 'Tall enough']

  1. Write a function that takes in a numeric grade (0 to 100) and returns the letter grade (no pluses or minuses). In a traditional rubric, that would be as follows:

  • 90 or above = A

  • 80 - 89 = B

  • 70 - 79 = C

  • 60 - 69 = D

  • 59 or below = F

  • Example: letter_grade(85) == 'B'

  1. Write another function that takes a list of numeric grades and returns their letter grades. Use the function written above in your answer.

  • Example: letter_grades([85, 62, 90]) == ['B', 'D', 'A']
